onPHP

> главная < примеры, возможности, чаво, скачать, пользователи, API документация, svn, websvn

о продукте

onPHP – это зрелый, лицензированный под GPL, многоцелевой объектно-ориентированный инструментарий для языка PHP.

новости

2005-11-04:
Появился архив почтовых рассылок.

0.2.4: реализована пессимистичная модель кэша в SmartDAO, работа над ошибками.

Поддержка MySQL вернулась из инкубатора.
Более детальный список изменений - там же, где и всегда.
